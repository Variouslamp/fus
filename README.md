# FUS (File Utility Suite)

> A personal file snapshot and synchronization tool built to explore versioning, storage, and system design concepts.

## Overview

FUS is a personal learning project inspired by version control systems such as Git, but focused on managing personal files instead of source code.

The goal is not to replace Git, cloud storage services, or backup software. Instead, FUS serves as a playground to understand how systems for versioning, synchronization, and storage work under the hood.

## Goals

* Learn how snapshot-based systems work.
* Understand hashing and file identification.
* Design a synchronization engine.
* Experiment with storage abstractions.
* Build a modular and extensible architecture.
* Learn by implementing, not by copying.

## Non-goals

The first versions of FUS will **not** include:

* Real-time synchronization.
* File diffing.
* Branches or merges.
* Collaboration features.
* Cloud provider integrations.

## Roadmap

* [ ] Scan directories.
* [ ] Generate a file index.
* [ ] Detect created, modified, and deleted files.
* [ ] Create snapshots.
